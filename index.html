Python Web Mapping Service (WMS) for visualizing geospatial data

System Requirements

  • >= 4GB RAM suggested, topology generation can impact system if low RAM

Configuration

Prerequisites

  • Python 2.7
  • HDF5 C library (dependency of netCDF)
  • libcURL (required for OPeNDAP)
  • netCDF C library (with OPeNDAP enabled)
  • libspatialindex (http://libspatialindex.github.com)
  • libPNG
  • libFreeType
  • libJPEG
  • libevent

Download

download the latest release and uncompress where you would like SCI-WMS to live

Install

install the Python dependencies using pip

if your NetCDF4 and HDF5 libraries are in non-typical locations, you will need to pass the locations to pip

NETCDF4_DIR=... HDF5_DIR=... pip ...
pip install -r requirements.txt

If you would like Gunicorn to use an alternative worker, the following are also supported:

  • eventlet
  • tornado

Test

test SCI-WMS is fully functional by running the following command

cd sciwms && python manage.py test

Start the services

start services on port 7000

gunicorn -c gunicorn_config_prod.py sciwms.wsgi:application

Gunicorn WSGI server configuration
